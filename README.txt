Objetivo

A proposta desse projeto é um sistema que liste o salário mínimo e a 
inflação do IPCA entre 2010 e 2020. Além disso, a aplicação deve listar
um comparativo do percentual de crescimento desses dois valores nesse mesmo período. 


Os seguintes conceitos usados no projeto:

- Variáveis
- Operadores
- Tipos de dados
- Estruturas condicionais
- Inputs
- Arrays
- Estruturas de repetição
- Manipulação de strings
